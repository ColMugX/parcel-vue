# parcel-vue

### Usage

This is a project template for [vue-cli](https://github.com/vuejs/vue-cli).

``` bash
$ npm install -g vue-cli
$ vue init ColMugX/parcel-vue my-project
$ cd my-project
$ npm install
$ npm run dev
```

### What's Included

- `npm run dev`: parcel

- `npm run build`: build with HTML/CSS/JS minification.
